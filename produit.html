One box to rule them all...

One Midi output connector
Three modes: Midi controller, Midi clock generator and Midi Timecode master
Main kit comes with 3 solid footswitches. Supports up to 7 total
Screen informs you of what has been sent
Midi Controller (default) mode:
  • Rotary encoder triggers Midi programs change
  • Long hold and short press of switch launches different events
Midi clock mode:
  • Rotary encoder changes BPM of clock signal (from 20 to 900 :) )
  • First switch reserved for start/stop/continue events
  • Second switch short press acts as tap-tempo
  • All other send same Midi CC as in default mode
MTC mode:
  • Rotary encoder scrubs in time
  • First switch reserved for start/stop events
  • All other send same Midi CC as in default mode
Works on 7V-12V DC with standard "Boss" adapter (positive outside, negative inside). You can purchase one here if you need to.
